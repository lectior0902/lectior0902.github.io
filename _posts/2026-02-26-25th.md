# 📌 QGIS 벡터 데이터 기본 조작

QGIS에서 가장 빈번하게 사용되는 **벡터 데이터(SHP, GeoPackage)** 로드 방법과 레이어 관리의 기초를 정리함.

---

# ✅ 벡터 레이어 추가 (SHP 파일)

ESRI 셰이프파일(.shp)을 로드하고 레이어 패널에 추가하는 과정

### 1️⃣ 데이터 로드 절차
- **원본 관리자 실행**: 툴바에서 [데이터 원본 관리자 열기] 클릭.
- **벡터 탭 설정**: [벡터] 탭 이동 후 원본 유형 '파일' 선택.
- **파일 경로 지정**: `...` 버튼 클릭 후 `korea_outline.shp` 선택 및 [추가].
- **결과**: 레이어 패널에 레이어 생성 및 맵 캔버스에 지도 표시됨.
---

korea_outline.shp파일을 레이어로 추가하기
1.	툴바 데이터 원본 관리자를 실행한다.
2.	벡터 탭을 클릭한다.
3.	원본 유형에서 파일을 선택한다.
4.	원본에서 ...버튼을 클릭한다.
5.	데이터가 있는 폴더로 이동한 후 korea_outline.shp파일을 선택한다.
6.	열기를 클릭하면 파일경로로 채워진 원본 대화창이 보인다.
7.	추가를 클릭하고, 닫기를 클릭한다.
8.	레이어 패널에는 korea_outline 레이어가, 맵 캔버스에는 지도가 표시된다.

# ✅ GeoPackage 파일 활용

단일 파일 내에 다수의 벡터/래스터 레이어를 저장하는 GeoPackage(.gpkg) 포맷 실습.

### 1️⃣ 레이어 연결 및 로드
- **GeoPackage 탭**: 데이터 원본 관리자에서 [GeoPackage] 선택.
- **새 연결 생성**: [새로 생성] 클릭 후 `training_data.gpkg` 파일 선택.
- **데이터 로드**: [연결] 버튼 클릭 후 `roads`, `buildings` 레이어 선택하여 추가.

### 2️⃣ 프로젝트 저장
- 레이어 구성 완료 후 **[프로젝트] -> [저장]**을 통해 `.qgz` 확장자로 현재 환경 저장.

---
GeoPackage 파일 열기
실습데이터 training_data.gpkg파일을 다운로드한다.
GeoPackage에서 레이어를 추가하기 위해서 GeoPackage와의 연결을 생성해야 한다.
1. 툴바의   [데이터소스관리자열기] 버튼을 클릭한다.
2.  GeoPackage탭을 선택한다.
3. [새로생성]을 클릭한 다음 실습파일 training_data.gpkg파일을 선택한다.
4. GeoPackage연결목록에 해당 파일의 경로가 추가되고, 드롭다운 메뉴에 표시된다.
5. [연결]버튼을 클릭한다. 아래 테이블 목록에 GeoPackage에 포함된 모든 레이어들의 목록을 확인할 수 있다.
6. roads레이어를 선택한 다음 추가 버튼을 클릭한다.
7. buildings레이어를 선택한 다음 추가 버튼을 클릭한다.
8. 닫기 버튼을 클릭한다.

# 📊 속성 테이블 활용 및 데이터 추출

지리 객체에 연결된 속성 정보를 질의하고 별도 데이터로 추출하는 과정.

### 1️⃣ 속성 테이블과 레이어 상호작용
- **탐색**: 레이어 선택 후 **F6** 입력. 테이블 행 클릭 시 지도상의 객체가 노란색으로 강조됨.
- **인코딩 수정**: 한글 깨짐 발생 시 [레이어 속성] -> [원본] -> [데이터 소스 인코딩]을 `EUC-KR`로 변경.

### 2️⃣ 조건부 데이터 추출 (표현식 활용)
- **표현식 선택**: 선택 툴바에서 [표현식으로 객체 선택] 아이콘 클릭.
- **질의문 작성**: `"SIGUNGU_NM" = '중구' 입력 후 [객체 선택].
- **별도 파일 저장**: 레이어 우클릭 -> [내보내기] -> **[선택한 객체를 다른 이름으로 저장]**.
- **저장 옵션**: 포맷은 `ESRI Shapefile`, 파일명은 `bnd_중구.shp`로 경로 포함하여 지정.

---
1.	해당 shp파일을 레이어 추가한다.
2.	선택툴바의 [표현식으로 객체 선택]툴을 클릭한다.
3.	표현식으로 선택 대화창의 [필드와 값]을 더블클릭하여 그룹을 펼친다.
4.	질의하고자 하는 필드("SIGUNGU_NM")를 더블클릭, 등호(=) 입력하고 오른쪽의 [모든 유일값] 버튼을 클릭하면 나타나는 목록에서 '중구'를 더블클릭하여 표현식 빈 칸에 "SIGUNGU_NM" = '중구' 가 입력되도록 한다.
5.	[객체 선택]버튼을 클릭한다.
6.	5. 레이어 패널의 우클릭 메뉴->[Export]->[선택한 객체를 다른이름으로 저장]을 클릭한다. 포맷을 확인하고 파일이름(bnd_중구.shp)을 경로명과 함께 지정한다.

# 🎨 인구 단위 단계 구분 시각화

속성값의 크기에 따라 지도의 심볼을 차등 적용하는 단계 구분(Graduated) 실습.

### 1️⃣ 실습 절차

- **대상 레이어**: `3w_seoul_dong.shp` 로드 및 속성 데이터 확인.

- **심볼 설정**: [속성] -> [심볼] -> 상단 드롭다운에서 **[단계 구분(Graduated)]** 선택.

### 2️⃣ 세부 파라미터 설정

- **값(Value)**: 인구수가 포함된 숫자 필드 선택.

- **색상 램프**: 밀도 표현에 적합한 그라데이션 선택.

- **모드(Mode)**: `자연스러운 분절(Jenks)` 또는 `등간격` 선택.

- **분류**: 하단 [분류] 버튼 클릭 후 [확인] 적용.


---
1.  3w_seoul_dong.shp 파일을 레이어로 추가
2. 필드 파악: F6을 눌러 속성 테이블을 열고, 인구 데이터가 들어있는 필드명을 확인
3. 심볼 설정: 레이어 패널에서 해당 레이어 우클릭 -> [속성] -> [심볼] 탭.
4. 단계 구분: 맨 위 드롭다운을 [단일 심볼]에서 [단계 구분(Graduated)]으로 변경.

•	값(Value): 인구수가 들어있는 필드 선택.

•	색상 램프: 원하는 색상 선택.

•	모드: [등간격] 또는 [자연스러운 분절] 선택. (자연스러운 분절(jerks) 선택함)

적용: 하단의 [분류] 버튼을 누른 후 [확인]을 클릭. 인구수에 따라 동네 색깔이 다르게 표시

# 💡 실습 팁 (Troubleshooting)

| 문제 상황 | 해결 방법 |
|:---|:---|
| 지도가 화면에 보이지 않음 | 레이어 우클릭 후 **[레이어 범위로 확대]** 클릭 |
| 특정 레이어가 가려짐 | 레이어 패널에서 드래그하여 점 > 선 > 면 순서로 배치 |
| 한글 깨짐 현상 | 데이터 소스 인코딩을 `System` 또는 `EUC-KR`로 변경 |
