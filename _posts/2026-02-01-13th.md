## **1. % 연산자**


```python
# 데이터 타입(%s: 문자열, %d: 정수)
# %를 문자 그대로 출력하고 싶을 때는 %%라고 두 번 써줘야 함

print("1. %% 연산자: %s님, %d단계 %s 입문 과정입니다." % (name, level, language))
```

    1. % 연산자: 초보 개발자님, 1단계 Python 입문 과정입니다.



```python
# %f: 실수 출력

level2 = 1.5

print("1. %% 연산자: %s님, %f단계 %s 입문 과정입니다." % (name, level2, language))
```

    1. % 연산자: 초보 개발자님, 1.500000단계 Python 입문 과정입니다.



```python
# %.1f 를 사용해서 소수점 한 자리만 출력

```

    1. % 연산자: 초보 개발자님, 1.5단계 Python 입문 과정입니다.

#**문자열 다루기 연습**

##**1. 문자열 인덱싱/슬라이싱**


```python
# 문자열
sentence = "Python Programming is Fun!"

print("--- 인덱싱 & 슬라이싱 ---")

# 인덱싱: 인덱스로 지정
print(f"1. 첫번째 글자: {sentence[0]}")           # P
print(f"2. 마지막 글자: {sentence[-1]}")          # !

# 슬라이싱: 범위로 잘라오기 [시작:끝+1]
print(f"2-1. 앞에서부터 6개: {sentence[0:6]}")      # Python
print(f"2-2. 중간: {sentence[7:18]}")               # Programming
print(f"2-3. 처음부터 끝까지: {sentence[:]}")
```

    --- 인덱싱 & 슬라이싱 ---
    1. 첫번째 글자: P
    2. 마지막 글자: !
    2-1. 앞에서부터 6개: Python
    2-2. 중간: Programming
    2-3. 처음부터 끝까지: Python Programming is Fun!


## **2. 문자열 메서드 사용(upper, lower, strip)**


```python
# 문자열 (공백 포함)
sentence = "   Hello World   "
```


```python
print(sentence)

# strip(): 양쪽 공백 제거
strip_sentence = sentence.strip()
print(f"strip(공백 제거): {strip_sentence}")

# upper() / lower(): 대문자/소문자 변환
print(f"upper: {strip_sentence.upper()}")
print(f"lower: {strip_sentence.lower()}")

# replace(): 문자 바꾸기
print(f"단어 교체: {strip_sentence.replace('Hello', 'Goodbye')}")

# split(): 문자열 나누기 (리스트로 반환)
print(f"단어별로 나누기: {strip_sentence.split(' ')}")
```

       Hello World   
    strip(공백 제거): Hello World
    upper: HELLO WORLD
    lower: hello world
    단어 교체: Goodbye World
    단어별로 나누기: ['Hello', 'World']


#**리스트 컴프리헨션**

##**1. 생성 및 변형**


```python
# 1부터 10까지 숫자의 제곱 리스트 만들기

squares = [x**2 for x in range(1, 11)]
print(f"제곱 리스트: {squares}")



# 주어진 문자열 모두 대문자 리스트로 만들기

cities = ["seoul", "gwangju", "busan"]
upper_cities = [i.upper() for i in cities]
print(f"대문자 리스트: {upper_cities}")
```

    제곱 리스트: [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
    대문자 리스트: ['SEOUL', 'GWANGJU', 'BUSAN']


##**2. 필터링**


```python
# 1~10까지의 수에서 홀수만 골라내서 그 값에 10을 곱한 리스트 만들기

result = [n * 10 for n in range(1, 11) if n % 2 != 0]
print(result)

# 이름 리스트에서 글자수가 4글자 이상인 이름 리스트 만들기
names = ["Lee", "Kim", "Park", "Choi", "Jung", "Kang"]

result = [i for i in names if len(i) >= 4]
print(f"이름 리스트: {result}")
```

    [10, 30, 50, 70, 90]
    이름 리스트: ['Park', 'Choi', 'Jung', 'Kang']


#**리스트와 딕셔너리 사용하기**

##**리스트**


```python
# 리스트 생성
fruits = ["apple", "banana", "cherry"]



#1. 인덱싱과 슬라이싱
print(f"원본 리스트: {fruits}")
print(f"첫 번째 두 개: {fruits[0:2]}")
print(f"마지막 두 개: {fruits[-2:]}")
print(f"모두: {fruits[:]}")



# 2. 추가/삭제

# 1. append(): 맨 뒤에, 2. insert(): 위치 지정
appended = fruits.append("orange")
print(fruits)

inserted = fruits.insert(1, "mango")    # 1번에 망고
print(fruits)

# 3. remove(): 요소 지정, 4. pop(): 위치 지정
removed = fruits.remove("apple")        # "apple" 삭제
print(fruits)


popped = fruits.pop(2)                  # 2번 삭제
print(fruits)
```

    원본 리스트: ['apple', 'banana', 'cherry']
    첫 번째 두 개: ['apple', 'banana']
    마지막 두 개: ['banana', 'cherry']
    모두: ['apple', 'banana', 'cherry']
    ['apple', 'banana', 'cherry', 'orange']
    ['apple', 'mango', 'banana', 'cherry', 'orange']
    ['mango', 'banana', 'cherry', 'orange']
    ['mango', 'banana', 'orange']


##**딕셔너리**


```python
# 딕셔너리 생성(제미나이 생성)
dev_info = {"name": "초보 개발자", "level": 1, "language": "Python"}
print(dev_info)


# 1. 데이터 수정
dev_info["level"] = 2                               # 값 수정
print(f"데이터 수정: {dev_info}")


# 2. 새로운 값 추가
dev_info["is_genius"] = True
print(f"새로운 값 추가: {dev_info}")


# 3. 삭제
del dev_info["is_genius"]
print(f"삭제: {dev_info}")


# 4. 키와 값을 모두 출력
print(f"모든 키: {list(dev_info.keys())}")
print(f"모든 값: {list(dev_info.values())}")
```

    {'name': '초보 개발자', 'level': 1, 'language': 'Python'}
    데이터 수정: {'name': '초보 개발자', 'level': 2, 'language': 'Python'}
    새로운 값 추가: {'name': '초보 개발자', 'level': 2, 'language': 'Python', 'is_genius': True}
    삭제: {'name': '초보 개발자', 'level': 2, 'language': 'Python'}
    모든 키: ['name', 'level', 'language']
    모든 값: ['초보 개발자', 2, 'Python']
